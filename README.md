# CSGO-YOLOv5

## 简单介绍

使用YOLOv5目标检测技术进行CSGO游戏中的辅助瞄准等。

此项目完全开源，且会提供不定期更新。

## 使用方法

1. 在aim_csgo文件夹下新建models文件夹，将训练好的YOLOv5系列模型放在里面。
2. 可使用export.py自行转换模型（可以但没必要）。
3. 剩下内容请看main.py中文字说明。
